Perguntas Frequentes

Como os dados são atualizados?

Nossos dados são atualizados em tempo real através de APIs confiáveis como CoinGecko e ExchangeRate-API, garantindo as cotações mais precisas.

As taxas incluem taxas de corretora?

Não, as taxas mostradas são apenas as cotações de mercado. Corretoras podem aplicar taxas adicionais nas operações reais.

Qual a precisão das cotações?

Para moedas tradicionais, trabalhamos com precisão de 2 casas decimais. Para criptomoedas, utilizamos 8 casas decimais para maior exatidão.

É seguro usar esse site?

Sim! Nosso site é apenas informativo e não realiza transações financeiras. Utilizamos apenas APIs seguras e confiáveis para obter os dados.
